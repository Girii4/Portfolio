Welcome to my portfolio! Engineered with a futuristic, cybernetic Stark OS and Jarvis theme, this project serves as a showcase of modern, high-performance web development and creative coding. 
🔗 **Live Experience:** [https://girii4.github.io/Portfolio/](https://girii4.github.io/Portfolio/)
---
## ⚡ Technical Highlights & Features
- **3D Arc Reactor Background:** A fully custom particle field representing an Arc Reactor, built with **Three.js** and designed to interact with mouse movement.
- **Micro-Animations & Smooth Motion:** Powered by **GSAP (GreenSock)** and **ScrollTrigger** for high-frequency, physics-based scroll transitions and interactive state entries.
- **Sleek Stark OS UI:** Designed using modern CSS techniques, featuring dynamic glassmorphism panels, futuristic typography (Inter & Orbitron), and interactive authorization modules.
- **Glowing Custom Cursor:** A dual-stage custom cursor and follower that provides tactile visual feedback.
- **Fully Responsive:** Beautifully adapts to mobile, tablet, and desktop screens with semantic HTML.
---
## 🛠️ Tech Stack
- **Core:** Semantic HTML5, Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid), ES6+ JavaScript.
- **3D Graphics:** [Three.js](https://threejs.org/) (WebGL Canvas rendering).
- **Animations:** [GSAP (GreenSock)](https://greensock.com/gsap/) & [ScrollTrigger](https://greensock.com/scrolltrigger/).
- **Icons:** FontAwesome v6.
